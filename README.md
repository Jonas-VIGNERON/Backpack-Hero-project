# Academic Java Game Project

This project was developed as part of an academic assignment in Java.

## My Contribution

I was responsible for designing and implementing several core game systems:

### Core Systems
- **Hero system** – creation and management
- **Monster system** – creation and behavior handling
- **Equipment system** – creation, bonus management, effect application, rotation mechanics
- **Effect system** – architecture and management
- **Inventory (Backpack)** – management and interactions
- **Event system** – merchant and treasure encounters
- **Combat system** – game logic implementation

### Focus
My work mainly focused on:
- Object-oriented design  
- Game logic implementation  
- System interactions
