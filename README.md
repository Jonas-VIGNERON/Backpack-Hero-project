This project was developed as part of an academic assignment in Java.

I was responsible for designing and implementing several core systems of the game:
	-Hero creation and management
	-Monster creation and behavior handling
	-Equipment system (creation, bonus management, effect application, rotation mechanics)
	-Effect system architecture
	-Inventory (backpack) management
	-Event system (merchant and treasure encounters)
	-Combat system logic

My work mainly focused on object-oriented design, game logic implementation, and system interactions.
